Proyecto Discapacidad

el proyecto fue realizado para recaudo bogota y es la pagina donde las personas consultas si pueden acceder a este beneficio el cual es un beneficio economico que se otorga mensualmente para ser usado en el sistema de trasporte de bogota (SITP TRASMILENIO) el proyecto fue desarrollado en Django, usando BD MySQL y para la interfaz grafica (HTML, CSS, JavaScript, Bootstrap) la aplicacion tiene dos vistas principales una vista donde los usuarios por tipo y numero de documento realizan la consulta y la segunda en donde se realiza la actualizacion de la informacion por tanto esta aplicacion maneja (sistema de logeo con permisos por usuario, envio de correo, generacion de archivos csv y xls entre otros)

Proyecto prevencion tailgating

Proyecto para la prevencion del tailgating(tecnica de ingenieria social) en exclusas peatonales, se capturaban imagenes utilizando una raspberry PI3 con una camara kinect y un algoritmo de redes neurales para el reconocimiento de personas, se almacena la informacion en una base de datos MYSQL, se hace uso del framework DJANGO para generar un api con los datos capturados sobre las alertas y los usuarios, para el consumo del api se trabaja HTML, CSS, y JQUERY para generar la interfaz de usuario)
